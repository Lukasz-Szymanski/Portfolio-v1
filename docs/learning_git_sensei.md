# Learning: Git-Sensei CLI (AI Tooling)

## Status
Projekt Zewnętrzny (Spotlight w Portfolio)

## O projekcie
Git-Sensei to narzędzie klasy "Meta-CLI", które rozwiązuje realny problem programistów: "Commit Fatigue" (zmęczenie pisaniem sensownych wiadomości commit). 

## Dlaczego ten projekt jest w portfolio?
Pokazuje on moje umiejętności wykraczające poza klasyczne aplikacje webowe:
1. **Automatyzacja CLI:** Budowa zaawansowanych narzędzi terminalowych w Pythonie.
2. **AI Prompt Engineering:** Projektowanie promptów, które analizują `git diff` i wyciągają intencje zmian kodu.
3. **Integracja z wieloma dostawcami:** Obsługa Gemini, Claude oraz lokalnych modeli przez Ollama.
4. **DevOps Mindset:** Narzędzie poprawiające kulturę pracy zespołu poprzez wymuszanie wysokiej jakości logów Git.

## Kluczowe wyzwania techniczne
- **Analiza różnic (Diffing):** Przetwarzanie surowego wyjścia komendy git w czytelny dla AI kontekst.
- **Privacy First:** Implementacja trybu offline i wsparcia dla lokalnych modeli, co jest kluczowe w pracy z wrażliwym kodem firmowym.
- **Konfiguracja TOML:** Stworzenie elastycznego systemu ustawień `.sensei.toml`.
